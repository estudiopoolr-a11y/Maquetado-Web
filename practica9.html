Lista de definiciones

Python
Un lenguaje de programación de alto nivel, interpretado y de propósito general.
Java
Un lenguaje de programación de propósito general, concurrente, orientado a objetos y basado en clases.
C++
Un lenguaje de programación de propósito general que es una extensión del lenguaje C.
JavaScript
Un lenguaje de programación interpretado, orientado a objetos y basado en prototipos.
PHP
Un lenguaje de programación de código abierto, especialmente adecuado para el desarrollo web y que puede ser incrustado en HTML.

git init
(Sirve para iniciar el repositorio local)

git config -- local user.name "Daniel"
(Esto configura al nombre de usuario localmente)

git config -- local user.email "dhanymarth@gmail.com"
(Esto configura al email de usuario asociado a la cuenta github)

git branch -M main
(Cambiar el nombre de la rama de Master a Main)

git add .
(añade todos los archivos al repositorio)

git commit -m "versión de subida"
(Es la versión que usted esta subiendo al repo)

git push -u origin main
(Este empuja los cambios locales a la nube).

git status
(Comprueba el estado de la rama)

gitIremote add origin https://github.com/Dhaniel-05/maquetado_web.git
(Añade la dirección de su repositorio se hace o ejecuta una única vez).
